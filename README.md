# Wool & Wood Online Shop
A custom built online shop for a client, as they wanted a site that would be more suited to their needs, as opposed to the limited options that site builders may provide.


## Tools, Frameworks and More
This site was built on the cakePHP framework (4.3) [https://cakephp.org/], utilising the inbuilt plugins. Also utilised is the SASS library [https://sass-lang.com/] and Unsemantic UI CSS grid [https://unsemantic.com/]. Unsemantic has been modified with a shortened extension, to reduce the length of class names. 
For transactions, this is undecided at the current point.

## Issues and PR's
For any issue that comes up, please when creating an issue give as much detail as possible, steps to reproduce, errors or similar, so that they may be resolved in a quick and timely manner.
For PR's, a quick overview of what's been changed, why its been changed, and anything you may think is relevant to the approval of a PR
